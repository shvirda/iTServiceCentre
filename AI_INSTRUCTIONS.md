## 1. Запуск приложения

### Рекомендуемый способ запуска

Для запуска приложения используй:

```powershell
C:\Python38_32\python.exe run_tk.py
```

**Важно:** На данном компьютере используется **только** Python 3.8 в `C:\Python38_32`!

**Все команды с Python должны использовать полный путь:**

- `C:\Python38_32\python.exe` вместо `python`
- `C:\Python38_32\Scripts\pip.exe` вместо `pip`

### Альтернативные способы

- `C:\Python38_32\python.exe run_tk.py` — запуск с tkinter (рекомендуется, не требует зависимостей)
- `C:\Python38_32\python.exe run.py` — запуск с PyQt6 (если установлен)
- `C:\Python38_32\python.exe run_backend.py` — только backend

### Инструкция 2: Проблема с запуском

> "запускаю cmd файл запуска и ничего не происходит"

**Решение:** Файл `start_app.cmd` был исправлен — теперь он ищет Python в различных стандартных директориях. Если по-прежнему не работает, запускай напрямую в терминале:

```powershell
python run_tk.py
```
